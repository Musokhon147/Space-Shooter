# Space Shooter

A classic 2D space shooter game built with Unity. Navigate your ship through space, destroy enemies, and collect power-ups to survive as long as possible!

## Features

- **Player Movement**: Smooth 2D movement with wrap-around boundaries.
- **Combat**: Fire lasers to destroy incoming asteroids and enemies.
- **Power-ups**:
  - **Triple Shot**: Fires three lasers at once for wider coverage.
  - **Speed Boost**: Temporarily increases player movement speed.
  - **Shields**: Protects the player from one hit without losing a life.
- **Health System**: Player starts with 3 lives. Visual damage is shown on the ship as lives decrease.
- **Score System**: Earn points for destroying enemies.

## Controls

| Action | Key(s) |
| :--- | :--- |
| **Move** | `W`, `A`, `S`, `D` or `Arrow Keys` |
| **Fire** | `Spacebar` |

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/Musokhon147/Space-Shooter.git
    ```
2.  Open the project in **Unity Hub**.
3.  Open the `Game` scene from the `Assets/Scenes` folder.
4.  Press **Play** to start the game.
